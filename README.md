## - Outline
단수의 모임을 관리할 수 있는 어플리케이션. 어플리케이션 가입자는 쉽게 일정을 관리하고 공유할 수 있습니다.
공지사항, 출석, 설문조사 등 다양한 기능을 통해 편리하게 모임을 관리할 수 있습니다.

### 주요 기능
- 출석체크
- 설문조사
- 공지사항
